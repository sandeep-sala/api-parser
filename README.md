# API Parser

It is an API parser which parser any REST API.
GET/POST supported

## Installation

```bash
git clone https://github.com/sandeep-sala/api-parser.git
or
git clone https://github.com/sandeep-sala/api-parser.git <folder name>
```


## Execute

```
open index.html in browser
```
